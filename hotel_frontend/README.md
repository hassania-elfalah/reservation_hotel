# Hôtel Excellence - Système de Réservation

Ce projet est un système de gestion de réservations pour l'Hôtel Excellence.

## Fonctionnalités

- Consultation des chambres disponibles
- Système de réservation en ligne
- Interface d'administration pour la gestion des chambres et réservations
- Authentification sécurisée

## Technologies utilisées

- **Frontend**: Vite, React, TypeScript, Tailwind CSS, shadcn/ui
- **Backend**: Laravel API
- **Base de données**: MySQL

## Installation

1. Clonez le dépôt
2. Installez les dépendances : `npm install`
3. Configurez l'URL de l'API dans `.env`
4. Lancez le serveur de développement : `npm run dev`
